I. Avant propos

L’application PAM dans sa version 1  a comme objectif de :

Ø      recueillir semestriellement, auprès de filiales, les informations commerciales utiles concernant l’activité monétique,

Ø      gérer ces informations pour les filiales sous forme de données dans une base de données centralisée administrée par BHFM,

Ø      restituer à l’ensemble de BHFM l’information sous forme de rapports composés de tableaux et de graphiques,

pour permettre le suivi et le pilotage de l’activité monétique des filiales de BHFM. Cette application remplace les feuilles Excel qui ont été utilisées au cours de l’années 2010 pour le recueil des données de l’activité monétique de chaque filiale.

Cette application, accessible sur l’intranet de BHFM à l’adresse suivante : https://pam.bhfm.socgen/, est disponible pour l’ensemble des acteurs de la monétique de BHFM. Cette application est bilingue : français, anglais.

Les données et les rapports sont exclusivement réservés à un usage interne et ne doivent être ni transmis ni publiés à l’extérieur de la banque sauf décision des responsables de la banque.

Tous les utilisateurs ayant des droits d’usage de cette application accèdent à l’ensemble des rapports.

Le contenu de la base de données est sauvegardé tous les jours et peut être restauré en cas d’incident grave.

Support et aide : le point de contact pour les questions d’usage de l’application est l’adresse mail : Animation.Cards@socgen.com

Veuillez adresser toutes vos questions à BHFM par l’intermédiaire de cette adresse.

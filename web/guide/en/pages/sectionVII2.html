2) Creating a new record: Add new data function

    • Display the table where you want to create a new record.

    • ATM acquiring activity example :

Clik on the button  :   Add new data

  • The form to capture data is displayed.

  • Give values to year, semester and acquiring channel data, then type values in the field’s boxes.

  • Use on line HELP by clicking the icon :

  • After updating values, three buttons are usable:

    • Update : confirm the changes and update the database,

    • Cancel : cancel the updates and display record with the previous values,

    • Abort: cancel the update process and go back to the table screen.

  • Only the administrator has the right to delete records in the database. If the removal of records in the subsidiary tables is required, the subsidiary should contact by mail the administrator to perform this operation.

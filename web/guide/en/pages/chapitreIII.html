III. Fonctions

A set of functions have been implemented in the program with the following features:

Record creation: writers only to capture every semester subsidiary cards business data.

Updating data: writers only to update or to correct records belonging to the subsidiary.

Viewing record: writers and readers to view detailed record belonging to subsidiary. Writers and readers cannot view records managed by the other subsidiaries.

Deleting record: administrator only. The subsidiaries have to ask BHM administrator to delete records.

Importing data process: writers only to import data using CSV files.

Reporting: all users to access the reports list and to issue cards business tables and graphics covering all the BHFM subsidiaries.

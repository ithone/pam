VI. Welcome page


The menu in the left column provides users with domains and functions defined as following:

  • Commercial Domain : capture, manage data and report by information type 

    • ATM POS Portfolio

    • CARDS Portfolio

    • MERCHANT Portfolio 

    • POS Acquisition 

    • ATM Acquisition 

    • Reporting : all available reports on the card business data

    • IMPORT : import function for CSV file to update data base.



  • User profile: user profile features connected and using the program: ID code, bank, area and country.


  • DISCONNECT: close the session.

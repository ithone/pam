3) Updating a record: EDIT function


    • EDIT function linked to each line/record in the table can display detailed data of the record in order to modify and update values of the record.

    • To display others records Click on Next Records or Previous Records buttons.

    • ATM-POS portfolio example :


  • After updating values, three buttons are usable:

    • Update : confirm the changes and update the database,

    • Cancel : cancel the updates and display record with the previous values,

    • Abort: cancel the update process and go back to the table screen.

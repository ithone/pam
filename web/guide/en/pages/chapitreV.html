V. Connection and application access

PAM program is available on BHFM intranet. The site address is: https://pam.bhfm.socgen/


Internet Explorer (upper than IE-6) and Firefox are recommended to be used in order to have all functions and features available.

User ID and password have to be provided by the user in the screen connection to access program. User ID and password are managed by BHFM administrator.

All issues to access the intranet site, to renew access code, to display screens have to be sent to BHFM administrator using this mail box: Animation.Cards@socgen.com
